# AI Student Performance Prediction

This project predicts student final exam scores using Machine Learning.

Technologies Used:
- Python
- Pandas
- Scikit-learn
- Flask
- HTML / CSS

Project Workflow:
1. Load student dataset
2. Train Linear Regression model
3. Save trained model
4. Use Flask web app for prediction

How to Run:

1 Install dependencies

pip install pandas scikit-learn flask

2 Train model

cd model
python train_model.py

3 Run application

cd ../app
python app.py

4 Open browser

http://127.0.0.1:5000
